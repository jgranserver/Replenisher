# Replenisher
Updated to latest ApiVersion

## Overview
Replenish your world resources!

Currently supported:
* ore
* pots
* altars
* chests
* life crystals
* trees
* floating islands
* config file for auto replen

Usage:
/replen pots 10
/replen ore 10 chlorophyte
/replenreload

Permissions:
tshock.world.causeevents
